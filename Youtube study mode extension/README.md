# 📚 YouTube Study Mode - Chrome Extension

**The easiest way to stop wasting time on YouTube and actually study.**

---

## 🎯 What is this?

This extension turns YouTube from a **distraction machine** into a **focused study tool**.

It helps you:
- Hide all distracting videos, comments, shorts, and recommendations
- Use a Pomodoro timer (25 minutes study + 5 minutes break)
- Block distracting pages while you're studying
- Track how much you actually studied every day
- Stay motivated with streaks and nice messages

---

## 🚀 How to Install (Super Simple Steps)

### Step 1: Download the files
Make sure you have all the files inside one folder called `youtube-study-mode`.

### Step 2: Open Chrome Extensions
1. Open Google Chrome
2. In the address bar, type this and press Enter:  
   **`chrome://extensions/`**

### Step 3: Turn on Developer Mode
Look at the top right corner → Turn **ON** the switch that says **"Developer mode"**

### Step 4: Load the Extension
1. Click the button **"Load unpacked"**
2. Select the folder `youtube-study-mode`
3. Click **Select Folder**

✅ Done! You should now see "YouTube Study Mode" in your extensions.

### Step 5: Pin the Extension (Recommended)
1. Click the puzzle icon (🧩) near the top right of Chrome
2. Find **YouTube Study Mode**
3. Click the pin icon so it always shows

---

## How to Use (Very Easy)

### 1. Turn on Focus Mode
1. Click the 📚 icon
2. Toggle **Focus Mode** to **ON**
3. Go to YouTube and refresh the page
4. All distractions (recommended videos, comments, shorts, etc.) will disappear!

### 2. Use the Study Timer
1. Click the 📚 icon
2. Choose how long you want to study (15, 25, 45, or 60 minutes)
3. Click **"Start Session"**
4. Study! The timer will count down
5. When time is up, take a 5-minute break

**Tip**: You can close the popup while the timer is running — it will keep working!

### 3. During Study Session
- If you try to open Shorts, Trending, or other distracting pages → it will show a "Back to Focus" screen
- Stay on educational videos only!

3. **Use Focus Mode** every time you open YouTube
4. **Take real breaks** — stand up, drink water, stretch

## Troubleshooting (Common Problems)

**Problem**: Extension icon not showing  
**Solution**: Click puzzle icon → pin the extension

**Problem**: Timer not working  
**Solution**: Make sure the extension is enabled and reload it

**Problem**: Focus mode not hiding videos  
**Solution**: Refresh YouTube page after turning on Focus Mode

**Problem**: Can't load the extension  
**Solution**: Make sure Developer mode is turned ON
