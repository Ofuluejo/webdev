# 🚀 How to Install YouTube Study Mode (Super Easy Guide)

This guide is written for **absolute beginners** — even if you’ve never installed an extension before.

---

## Step-by-Step Installation

### Step 1: Prepare the Folder
1. Make sure you have all the extension files inside one folder named **`youtube-study-mode`**
2. Do **not** open any file — just keep them together in that folder

---

### Step 2: Open Chrome Extensions Page
1. Open **Google Chrome** browser
2. In the address bar at the top, type this exactly and press **Enter**:


---

### Step 3: Enable Developer Mode
1. Look at the **top right corner** of the page
2. You will see a switch labeled **"Developer mode"**
3. Click the switch to turn it **ON** (it should turn blue)

---

### Step 4: Load the Extension
1. Click the big blue button that says **"Load unpacked"**
2. A file window will open
3. Navigate to and select the **`youtube-study-mode`** folder
4. Click **Select Folder** (or Open)

---

### Step 5: Pin the Extension (Very Important)
1. Look at the top right of your Chrome browser (near the three dots)
2. Click the **puzzle piece icon** 🧩
3. Find **"YouTube Study Mode"**
4. Click the **pin icon** next to it so it stays visible

---

## ✅ You’re Done!

You should now see the **📚** icon in your toolbar.

---

## First Time Setup (Test It)

1. Click the **📚** icon
2. Toggle **Focus Mode** to **ON**
3. Go to [youtube.com](https://www.youtube.com) and **refresh** the page
4. Try opening Shorts or the homepage — you should see distractions hidden!
5. Try the timer — click **Start Session**

---

## Common Problems & Fixes

| Problem | Solution |
|-------|---------|
| "Load unpacked" button is gray | Make sure **Developer mode** is turned ON |
| Extension icon not visible | Click puzzle icon → pin the extension |
| Focus mode not working | Turn Focus Mode ON → Refresh YouTube page |
| Timer not counting | Reload the extension from `chrome://extensions/` |
| Extension not working | Click the **Refresh** icon (↻) next to the extension name |

---

## How to Update the Extension Later

If I give you new files in the future:
1. Replace the old files with new ones in the same folder
2. Go to `chrome://extensions/`
3. Click the **Refresh** button (↻) on "YouTube Study Mode"

---

## How to Uninstall

1. Go to `chrome://extensions/`
2. Find **YouTube Study Mode**
3. Click **Remove**
4. Click **Remove** again to confirm

---
